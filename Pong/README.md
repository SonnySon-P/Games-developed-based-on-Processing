# Pong

**作品名稱：** 反彈球
	
**動機：** 本次遊戲的製作，主要懷念過往遊戲機的小遊戲，興起模仿該遊戲方式所撰寫。

**開發環境：** Processing(該IDE以Java為撰寫語言)

**遊戲解說：** 進行遊戲過程，遊戲畫面會出現一顆移動中小紅球，玩家必須移動遊戲畫面底下的黑色板子左右移動，藉由反彈小紅球，不讓其落入黑色板子區塊。
