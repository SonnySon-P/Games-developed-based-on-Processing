# PacMan

**作品名稱：** 小精靈
	
**動機：** 本次遊戲的製作，主要懷念過往遊戲機的小遊戲，興起模仿該遊戲方式所撰寫。

**開發環境：** Processing(該IDE以Java為撰寫語言)

**遊戲解說：** 遊戲過程中，必須不被鬼給碰觸到，否則將會結束遊戲。而過關的條件，便是將畫面中的所有黃色星星都吃完。
